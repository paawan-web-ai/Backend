API ENDPOINTS
-----------------

1. CREATE POST
   Route: POST /api/posts
   Body: multipart/form-data
     - title (string)
     - description (string)
     - image (file) - uploaded via multer
   Process:
     1. Multer captures image in memory
     2. ImageKit uploads image to cloud
     3. Post created in MongoDB
   Response: 201 { message, post }

2. GET ALL POSTS
   Route: GET /api/posts
   Response: 200 { message, posts: [Post] }

3. DELETE POST
   Route: DELETE /api/posts/:id
   Response: 200 { message }

4. UPDATE POST (PATCH)
   Route: PATCH /api/posts/:id
   Body: { description: string }
   Response: 200 { message }
